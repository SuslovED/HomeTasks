# Tiger & Keeper

Реализация задачи с использованием разделяемой памяти и семафоров.

## Описание

В программе участвуют два процесса:

- **Смотритель** — наполняет общую «миску» мясом  
- **Тигр** — забирает мясо из миски  

Семафор обеспечивает взаимное исключение при доступе к общей памяти.

## Сборка и запуск

Сборка:
make

Запуск:
./tiger_keeper

Остановка:
Ctrl + C

## Очистка ресурсов

При завершении работы программа корректно удаляет:
- разделяемую память
- семафор

## Основные функции

`cleanup()`  
Удаляет семафор и разделяемую память.

`handle_sigint()`  
Обрабатывает сигнал `SIGINT` (Ctrl+C) и вызывает `cleanup()`.

`semaphore_wait()`  
Блокирует доступ к миске (захват семафора).

`semaphore_post()`  
Разблокирует доступ к миске (освобождение семафора).

## Логика работы

1. Смотритель захватывает семафор и наполняет миску
2. Семафор освобождается
3. Тигр захватывает семафор и ест мясо
4. Семафор освобождается
5. Процесс повторяется
